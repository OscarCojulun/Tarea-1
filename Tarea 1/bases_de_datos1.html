Bases de Datos 1

Temas: Triggers, Procedimientos, Funciones y Sentencias SQL

En el curso de Bases de Datos 1 se estudian los elementos fundamentales para el manejo avanzado de datos mediante SQL. Entre los temas más importantes se encuentran los triggers, procedimientos almacenados, funciones, y comandos como INSERT, DELETE, UPDATE y ALTER. Estas herramientas permiten automatizar tareas, validar reglas de negocio y mantener la integridad de los datos.

🔑 Puntos Clave

✔️ Triggers: Son activadores automáticos que se ejecutan ante eventos como inserciones o eliminaciones.
✔️ Procedimientos: Conjunto de instrucciones SQL almacenadas que se pueden ejecutar bajo demanda.
✔️ Funciones: Similar a los procedimientos, pero devuelven un valor y se pueden usar en consultas.
✔️ INSERT: Se utiliza para agregar registros a una tabla.
✔️ DELETE: Elimina uno o más registros existentes.
✔️ UPDATE: Modifica valores existentes en los registros.
✔️ ALTER: Permite cambiar la estructura de una tabla (añadir columnas, modificar tipos, etc.).

📋 Ejemplos de Sentencias SQL

Comando Ejemplo
INSERT INSERT INTO alumnos (nombre, edad) VALUES ('Ana', 21);
DELETE DELETE FROM alumnos WHERE edad < 18;
UPDATE UPDATE alumnos SET edad = 22 WHERE nombre = 'Ana';
ALTER ALTER TABLE alumnos ADD COLUMN correo VARCHAR(50);
